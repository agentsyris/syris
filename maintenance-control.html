Quick Actions:

Manual Control:

To manually switch between modes:

  1. Enable Maintenance Mode: Rename index.html to index-full.html and rename coming-soon.html to index.html
  2. Disable Maintenance Mode: Rename index.html to coming-soon.html and rename index-full.html to index.html

Environment Variables (for Vercel/Netlify):

Add this environment variable to your hosting platform:

MAINTENANCE_MODE=true

Set to false to show the full site, true for maintenance mode.
